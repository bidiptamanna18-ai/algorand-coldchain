# Algorand Cold Chain

## Overview
Algorand Cold Chain is an IoT-powered monitoring system that tracks temperature-sensitive goods such as vaccines, medicines, and food products.

The project combines IoT sensors with the Algorand blockchain to create secure and tamper-proof temperature records.

## Problem
Cold chain products can become unsafe when exposed to temperatures outside their required range.

Traditional monitoring systems can be altered or manipulated, making it difficult to verify product safety.

## Solution
This project collects temperature data from IoT devices and stores verification records on the Algorand blockchain.

## Features
- Real-time temperature monitoring
- Alert generation for temperature violations
- Secure blockchain verification
- Transparent audit trail
